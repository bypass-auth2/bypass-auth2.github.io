bypass is a 2-factor-authentication solution for purchases through voice controlled devices such as Alexa or Google Home. Thanks to our solution we provide you with an easy to implement solution that provides a dynamically generated code that is sent to your phone via sms each time you have a request that needs protection and security.

bypass is your friend when it comes to 2-factor authentication

It is easy to implement. we have abstracted all of the difficult work using flowroute and providing an easy to use python module.

Its as easy as this flask tutorial

download the source code from github

bypass-auth

place the source code into your alexa skills directory

update your python modules using the requirements module

1. set up a virtualenvironment for your project

2. migrate to the bypass-auth folder

3. install the requirements

    $ pip install -r requirements.txt

4. retrieve your users phone number from your database (amazon linked accounts tutorial below)
https://developer.amazon.com/blogs/post/Tx3CX1ETRZZ2NPC/Alexa-Account-Linking-5-Steps-to-Seamlessly-Link-Your-Alexa-Skill-with-Login-wit

receiving linked accounts requires your skill to be accepted first, for development, you can hard code in a test number or use a dummy account. the xample below uses Flask-sqlalchemy


    user = User.query.filter_by(userid='#######').first()
    phone_number = user.phone_number

5. Add the code into your main routing file and save what the method returns

  pass_code = bypass_sms.send_auth(phone_number)

6. Configure your auth service to receive your variables in the config file

This includes your:

  • flowroute access key
  • flowroute secret key
  • flowroute number
    
    #configuration.py
    USERNAME = '##########'
    PASSWORD = '18###88ec#####d#########f7cab5'
    from_number = '15104050151'

Now to run!

$ python main.py
